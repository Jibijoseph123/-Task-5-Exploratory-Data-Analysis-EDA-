# Task 5: Exploratory Data Analysis (EDA)

## Objective
Performed exploratory data analysis on the Titanic dataset to identify patterns, trends, and relationships.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Analysis Performed
- Statistical summary using describe() and info()
- Univariate analysis (histograms, boxplots)
- Bivariate analysis (countplots, scatterplots)
- Correlation analysis using heatmap
- Multivariate analysis using pairplot

## Key Findings
- Gender and passenger class strongly influence survival.
- Fare shows positive correlation with survival.
- Dataset contains skewed distributions and outliers.
